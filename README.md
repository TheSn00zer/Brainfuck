# Brainfuck
This is an interpreter for [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck).

## Usage
### File
`java Brainfuck F examples/squares.b`

### User input
`java Brainfuck ++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.`
